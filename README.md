# STM32F0_Multi_Peripheral_PCB
This is a project of an STM32F0 PCB which is a mixed-signal board that includes multiple peripherals and the following components:

## Components:
-	STM32F0 MCU
-	 USB 2.0 connection
    o	With added ESD protection
-	3.3V power regulator LDO
-	Li-Ion/ Li-Pol battery charger
-	External flash memory
-	A GPIO extender
-	Accelerometer + Gyroscope IC
-	Motor Controller IC
-	RGB LED
-	Power-ON LED
-	SWD Connector

## Board Setup:
It is a 4-layer board with the following stack-up:

Layer: 1	SIGNAL
Layer: 2	GND
Layer: 3	POWER
Layer: 4	SIGNAL
